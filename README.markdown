
Introduction
============

Seafile-iOS is a the iOS client for [Seafile](http://www.seafile.com).

Build and Run
=============

Follow these steps :

	git clone https://github.com/haiwen/seafile-iOS.git
	cd seafile-iOS
	git submodule init
	git submodue update
	open seafile.xcodeproj

Then you can run seafile in xcode simulator.
